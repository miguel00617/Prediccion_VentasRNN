ForeCasting (LSTM): Predicción de Ventas

Este proyecto implementa un modelo de Red Neuronal Recurrente (LSTM) para predecir ventas semanales utilizando datos históricos de tiendas y departamentos.
La aplicación incluye una interfaz web interactiva desarrollada con Streamlit, donde el usuario puede seleccionar una tienda (Store) y un departamento (Dept) para generar predicciones de ventas futuras.
El objetivo es analizar patrones temporales y generar un forecast de ventas que ayude a la toma de decisiones.

Este proyecto se divide en:    
   1er: EDA(Análisis de Datos Exploratorio), Limpieza y transformación.
   2do: Preparación para el Modelo
   3ro: Creación del Modelo.
   4to: Entrenamiento
   5to: Evaluación
   6to: Conclusión

Tecnologías: Python(Pandas, Numpy, Matplotlib, seaborn, Scikit Learn, TensorFlow)
Fuente de Datos:  https://www.kaggle.com/datasets/aslanahmedov/walmart-sales-forecast
